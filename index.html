Project Features

The innovative features of our project are:


  1. Checkpoint:
    Rover can pass through a checkpoint before going to the destination.
  2. Multiple Destination:
    Rover will visit that destination which has the shortest path.
  3. Different types of obstacles:
    Users can add different types of obstacles such as walls, hills, craters, glaciers and storms.
  4. Different rates of algorithm run:
    Users can run the algorithm in slow, medium or fast speed.
Future Scope of our project

Features we wish to add in future are:

  • Add a real time graph which compares the running time of all algorithms.
